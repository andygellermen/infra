# Easy-Event-Planner – Admin UI Structure

## Navigation MVP 1

Dashboard, Events, Teilnehmer, Snippets, E-Mails, Datenschutz, Einstellungen.

Später zusätzlich: Warteliste, Einladungen & Rabatte, Zahlungen, Kalender, Zertifikate.

## Dashboard

Widgets: heutige Events, nächste Events, bestätigte Teilnehmer, Warteliste, freie Plätze, offene E-Mail-Jobs, letzter Datenschutzlauf.

## Events

Liste: Datum, Titel, Status, Reihe, Ort/Online, Teilnehmer, Warteliste, Sichtbarkeit, Aktionen.

Formularfelder: Titel, Slug, Reihe, Beschreibung, Start/Ende, Zeitzone, Ort, Adresse, Online-Link, Teilnahmeart, maximale Teilnehmerzahl, Anmeldung aktiv, Warteliste aktiv, sichtbar.

## Eventdetail

Tabs: Übersicht, Teilnehmer, Warteliste, E-Mails, Snippet, Kalender, Änderungen.

Aktionen: veröffentlichen, deaktivieren, bearbeiten, absagen, verschieben, Teilnehmerliste exportieren, Morgen-E-Mail testen.

## Teilnehmerliste

Spalten: Name, E-Mail, Telefon, Teilnahmeart, Status, Zahlungsstatus, angemeldet am, Aktionen.

Aktionen: Details, stornieren, als teilgenommen markieren, E-Mail senden.

## Snippet Builder

Felder: Name, Layout, Eventreihe, Limit, vergangene anzeigen, Anmeldebutton, Zielcontainer, Theme. Ausgabe ist kopierbarer Script-Code.

## Datenschutz

Zeigt Retention Policies, letzten Löschlauf, nächsten Lauf und Dry-Run-Ergebnis. Aktionen: Fristen bearbeiten, Dry-Run starten, Löschlauf starten.

## Einstellungen

Mandantenprofil, Branding, E-Mail-Absender, PayPal, Datenschutzkontakt, Zeitzone, Sprache, Nutzer & Rollen.

## UX-Regeln

Klare Sprache, kritische Aktionen mit Bestätigung, Snippet-Code immer kopierbar, Status visuell eindeutig, mobil nutzbar, Tabellen exportierbar vorbereiten.
